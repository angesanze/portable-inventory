Varasto SDK Demo

Three ways to embed the inventory widget. Each uses a different theme.

JavaScript API

JS SDK 
const widget = new PortableInventory({
  apiKey: 'demo',
  container: '#widget-js',
  theme: { primaryColor: '#3b82f6' }
});
widget.mount();

Web Component

Custom Element 
<portable-inventory
  api-key="demo"
  product-id="1"
  theme-primary="#10b981"
  theme-bg="#f0fdf4"
></portable-inventory>

Raw iframe

Legacy 
<iframe src="/widget?api_key=demo"
  style="width:100%; border:none;"
></iframe>
